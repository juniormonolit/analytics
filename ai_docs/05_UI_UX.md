# UI/UX правила

## Общий layout

```
AppShell
  Sidebar
  MainArea
    SectionHeader
    FilterBar
    ReportTabs
    ReportToolbar
    ReportTable
```

## Sidebar

Скрываемый сайдбар.

Пункты:

- Найм
- Маркетинг
- Продажи
  - По менеджерам
  - По товарным группам
- Реализация
- Настройки

Активный раздел подсвечивать.

## FilterBar

Располагается сверху в одну строку.

Порядок:

1. Период.
2. Отделы.
3. Остальные фильтры.
4. Кнопка сброса/применения, если понадобится.

Фильтры сохраняются на уровне раздела `sales`, а не на уровне таба.

## DateRangePicker

Поведение:

- поле показывает `29.04.2026 - 29.04.2026`;
- dropdown с календарем слева и пресетами справа;
- пресеты: Сегодня, Вчера, Эта неделя, Прошлая неделя, Этот месяц, Прошлый месяц;
- рядом в состоянии хранится comparisonPeriod.

## DepartmentTreeFilter

Требования:

- дерево отделов;
- чекбоксы;
- partial state для родителя, если выбрана часть детей;
- клик по стрелке раскрывает/скрывает;
- клик по чекбоксу выбирает узел со всеми детьми;
- в кнопке показывать `Отделы (7)`.

Если иерархии нет, временно показывать плоский список teams.

## ReportTabs

В разделе Продажи:

- По менеджерам
- По товарным группам

Табы переключают reportSlug, но не сбрасывают фильтры раздела.

## ReportToolbar

Содержит:

- выбор показателей;
- выбор набора показателей;
- группировка: Нет / Отдел / Итого;
- экспорт позже;
- обновить.

## MetricPickerModal

Две вкладки:

- Показатели;
- Наборы.

Вкладка Показатели:

- поиск;
- теги категорий;
- список метрик;
- drag handle для порядка;
- чекбокс или добавление/удаление;
- справа preview выбранных.

Вкладка Наборы:

- список пользовательских наборов;
- применить набор;
- удалить набор;
- позже создать набор из текущих выбранных метрик.

## Таблица

Требования:

- sticky header;
- sticky first column;
- горизонтальный скролл;
- итоговая строка;
- сортировка по колонкам;
- клик по строке открывает drill down;
- скелетон при загрузке;
- пустое состояние, если данных нет.

## DrillDownPanel

Открывается справа.

Заголовок:

- название строки;
- период;
- кнопка закрытия.

Содержимое:

- хлебные крошки уровней;
- таблица детализации;
- кнопка назад.
